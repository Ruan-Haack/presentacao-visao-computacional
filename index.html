01

Como a IA aprende a enxergar

  • A imagem é uma grade de pixels cada pixel, um número de intensidade.
  • Um kernel (matriz pequena, ex. 3×3) desliza sobre a imagem.
  • Em cada posição: multiplica e soma os valores sobrepostos.
  • O resultado realça bordas, texturas e padrões base de toda CNN - Convolutional Neural Network (Rede Neural Convolucional).
  • Camadas sucessivas aprendem conceitos cada vez mais abstratos: pixels → bordas → formas → objetos. 
# kernel de detecção de bordas
kernel = [[ 0, -1,  0],
          [-1,  4, -1],
          [ 0, -1,  0]]
saida = convolve(imagem, kernel)
02

Por que ainda é difícil Desafio #1

Para nós é óbvio. Para o modelo, ambiguidade pura.

Aparência similar

Texturas, cores e formas podem ser idênticas entre classes completamente diferentes.

Variação intra-classe

Um mesmo objeto (cadeira, cão, carro) varia enormemente de instância para instância.

Escala e perspectiva

O modelo deve reconhecer o mesmo objeto visto de perto, de longe, em ângulos diferentes.

02

Por que ainda é difícil Desafio #2

  • Perda de informação cromática: converter para escala de cinza elimina diferenças que o humano usa para separar objetos (fruta madura vs. verde, semáforo vermelho vs. amarelo).
  • Iluminação e ruído: a mesma cena à noite, em névoa ou com super-exposição gera distribuições de pixel completamente diferentes para o mesmo objeto.
  • Oclusão: quando parte do objeto é tapada, o modelo deve inferir o todo a partir do fragmento visível.
  • Solução parcial: treinar com data augmentation rotações, flips, variações de brilho para o modelo generalizar melhor.
03

Onde a Visão Computacional já está sendo usada

Saúde: detecção de tumores em raio-X e ressonâncias com precisão superior a radiologistas em alguns cenários.
Veículos autônomos: identificação de pedestres, sinais, faixas e obstáculos em tempo real.
Agricultura: drones monitoram lavouras detectando pragas e estimando produtividade por imagem.
Segurança: reconhecimento facial, detecção de intrusos e análise de comportamento em câmeras.
Indústria: controle de qualidade automático identifica defeitos em peças que o olho humano não percebe.
